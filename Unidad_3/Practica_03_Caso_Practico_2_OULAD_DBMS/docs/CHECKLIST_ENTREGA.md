# Checklist de entrega

## Artículo

- [x] Portada.
- [x] Resumen y palabras clave.
- [x] Abstract y keywords.
- [x] Tabla de contenido.
- [x] Introducción, metodología, hallazgos, discusión y conclusiones.
- [x] Citas narrativas, parentéticas y textual breve.
- [x] Referencias APA 7.
- [x] Anexos y división colaborativa.
- [x] Sustituir los resultados preliminares por la ejecución final de OULAD.
- [ ] Completar facilitador, integrantes, matrículas y fecha.

## Proyecto

- [x] POO y TAD colección (`ResultRegistry`).
- [x] OSEMN encapsulado.
- [x] EDA univariado, bivariado, dispersión, curtosis, box plot y correlación.
- [x] Auditoría y tratamiento de missing.
- [x] Codificación explícita de ordinales.
- [x] Tres algoritmos para variable dicotómica.
- [x] Tres algoritmos para variable ordinal.
- [x] Tres algoritmos para variable intervalo/razón.
- [x] CSV general y predicciones caso a caso.
- [x] F1 manual con TP, FP, TN y FN.
- [x] Precision macro, recall macro, F1 macro, accuracy y ROC-AUC.
- [x] MSE y R².
- [x] Matrices de confusión e importancias.
- [x] Rutas relativas; sin path local.
- [x] Suite ampliada de 24 escenarios automatizados.
- [x] Documentación HTML local y navegable.
- [x] Ejecutar el proyecto completo y conservar las salidas.
- [x] Preparar el paquete final y los enlaces de entrega.
