# Unidad 3: Análisis Exploratorio de Datos (EDA) y Estadística Inferencial

Asignatura: INF-8237-C2 Ciencia de Datos I  
Facilitador: Dr. Silverio del Orbe Abad

## Descripción

La Unidad 3 cubre el análisis exploratorio de datos (EDA) y estadística inferencial. Los estudiantes aprenderán a:

- Limpiar y preparar datos para análisis
- Realizar análisis descriptivos univariados y multivariados
- Visualizar patrones y relaciones en los datos
- Realizar pruebas de hipótesis estadísticas
- Interpretar resultados y comunicar hallazgos
