# Scripts auxiliares

Este folder queda reservado para automatizaciones generales de la Unidad 2.

Ejemplos de scripts futuros:

- Generar un paquete ZIP de entrega.
- Ejecutar pruebas de ambos casos.
- Copiar evidencias a anexos.
- Crear reporte de estructura del proyecto.

Script activo en Caso 2:

- `caso_practico_2_sakila_crud_orm/setup_run_sakila_docker.sh`: prepara MySQL/Sakila en Docker, prueba conexion y abre el menu CRUD/ORM.
- `generar_informe_caso2_pdf.py`: genera el informe APA/UASD en PDF para el Caso practico 2.
