# Agentes IA - UASDVirtual

Este folder documenta agentes especializados para apoyar el desarrollo de la Unidad 2. Los agentes no sustituyen la revision del equipo; sirven para acelerar analisis, propuestas, revision y documentacion.

## Agentes

- `sakila_backend_agent.md`: encargado del CRUD/ORM nativo.
- `oulad_data_agent.md`: encargado del modelo relacional, ETL y EDA.
- `apa_uasd_agent.md`: encargado del formato academico UASD/APA 7.
- `qa_reviewer_agent.md`: encargado de pruebas, riesgos y checklist final.

## Regla de uso

Cada agente debe producir recomendaciones verificables. Si genera codigo, el equipo debe ejecutarlo. Si propone citas, el equipo debe comprobar fuente, fecha, autor y correspondencia con la referencia.
