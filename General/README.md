# Material general

Este folder reúne archivos transversales o históricos que no pertenecen directamente a una
práctica de las unidades.

## Organización

### Documentos académicos

Versiones históricas y finales del ensayo sobre el origen y evolución de la ciencia de datos.

### Normas y plantillas APA

- Guía FIA/UASD de normas APA 7.
- Anexo de formato para trabajos.
- Plantilla académica.

### Recursos visuales

- Logo UASD.
- Línea de tiempo.
- Códigos QR y anexos gráficos.
- Recurso HTML histórico.

### Scripts generadores

Scripts usados para crear versiones DOCX, PDF y documentos con formato APA.

### Herramientas

Skill local de apoyo para formato UASD/APA 7 y sus referencias.

### Paquetes locales

Archivos ZIP preparados para entrega. Se mantienen fuera de Git mediante `.gitignore`.

## Privacidad

Los archivos de texto y código usan rutas relativas. No se documentan rutas absolutas de la
computadora de desarrollo.
